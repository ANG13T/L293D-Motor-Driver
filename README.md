# L293D-Motor-Driver

This projects lets you control two motors either in a clockwise or counterclockwise rotation with the help of L293D!

## Materials

- Breadboard
- 2 DC Motors
- Arduino Uno
- Around 15 - 20 jumper wires
- L293D
